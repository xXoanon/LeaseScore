# Car Lease Calculator

A professional, feature-rich car lease calculator that helps users determine if their lease deal is good, compare leasing vs buying, and understand payment breakdowns.

## Features

✅ **Smart Deal Evaluation** - Uses the 1% rule and industry best practices  
✅ **Lease vs Buy Comparison** - Side-by-side financial analysis  
✅ **Detailed Breakdowns** - Understand every component of your payment  
✅ **Simple & Advanced Modes** - Choose your level of detail  
✅ **Real-time Calculations** - Instant results as you type  
✅ **Mobile Responsive** - Works perfectly on all devices  
✅ **Money Factor or APR** - Flexible input options  
✅ **Ad-Ready** - Pre-configured ad slots for monetization  

## Quick Start

1. Open `index.html` in your browser
2. Enter your lease details
3. Get instant analysis and recommendations

## Usage

### Basic Usage
1. Enter MSRP and Negotiated Price
2. Input your monthly payment (before tax)
3. Set down payment, lease term, and sales tax
4. Choose Money Factor or Interest Rate
5. Click Calculate or let it auto-calculate

### Advanced Features
- Toggle **Advanced Mode** for additional fields (residual value, trade-in, fees)
- Switch to **Lease vs Buy** tab to compare financing options
- View **Payment Breakdown** for detailed cost analysis

## Customization

### Change Colors
Edit `styles.css` to modify the color scheme:
```css
/* Primary gradient */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add Your Donation Link
Edit `calculator.js` line ~380:
```javascript
<a href="YOUR_LINK_HERE" class="coffee-button" ...>
```

### Configure Ads
Replace ad placeholder divs in `index.html` with your ad network code.
